# Brief — Template de Arte do Rax (capa + carrossel) + botão Canva

> Spec dada pelo Rudá (16/jun/2026). Aplicar no gerador de arte do inbound/raccoon.

## Layout do card (capa/slide) — formato 1080×1350 (carrossel) e 1200×630 (capa blog)
- **Topo:** lockup branco = **logo EPI·USE + logo da LOB** (ex.: ServiceNow, SuccessFactors). **Logos GRANDES** (hoje estão pequenos demais no canto sup. direito — aumentar).
- **Centro:** headline curta **3–5 palavras** + **número GIGANTE** (Open Sans / Poppins **800**). O número é o herói visual.
- **Card escuro:** 1 linha de **contexto** + **fonte do dado** (etiqueta pequena).
- **Rodapé:** **selo ERP.ngo** (1% — conservação) + **epiuse.com.br**.
- Paleta: navy `#001844` + red `#cd1543` + branco. Fundo: foto com overlay navy OU navy sólido.

## Botão "Editar no Canva"
- Em cada arte gerada (capa/slide), botão **"Editar no Canva"** que **abre o Canva** (template da marca) pra Duda refinar.
- v1: abre Canva (deep-link template). v2 (futuro): Canva Connect/Autofill API pra popular o design com o conteúdo automaticamente (precisa app dev + brand template com campos).

## ServiceNow
- Deve estar contemplado como LOB no engine + ter lockup de logo. Post real de referência: carrossel "Otimize seu tempo com ServiceNow" (7 páginas) — headline "Você precisa de mais horas no dia... Ou de uma plataforma que otimize as que você já tem?".
