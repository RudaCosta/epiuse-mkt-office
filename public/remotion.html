🎬 Remotion — vídeos do Office em código

Remotion é um framework React pra gerar vídeos animados programaticamente. Em vez de editar no Premiere/After Effects, você escreve código (HTML/CSS/React) e o Remotion renderiza MP4/GIF/PNG sequence com FFmpeg. Tudo versionável, automatizável, integrável com nossas APIs reais.

⚙️ O que o Remotion permite tecnicamente
🎞️

Frame-by-frame control

Cada frame é uma chamada React · interpolate() + Easing em qualquer prop CSS

🖼️

Componentes React

Reusa nossos componentes HTML do Office direto · sem reinventar

🎵

Áudio + sincronia

Voiceover via TTS · trilha musical · efeitos sonoros sincronizados ao frame

📐

Qualquer formato

9:16 (TikTok) · 16:9 (YouTube) · 1:1 (Instagram) · 4:5 (LinkedIn)

🔁

Dados dinâmicos

calculateMetadata() puxa de API → cada render é único pro dado do dia

⚡

FFmpeg sob o capô

Render rápido em CPU · GPU acelera ainda mais · cloud render disponível

📺

Lottie + Rive

Anima vetorial avançada via @remotion/lottie e bibliotecas de motion

🎨

Tailwind + CSS

Estilização normal · só não usa transitions/animations CSS (frame controla)

💻 Como parece o código (exemplo do hover demo)
// Anima opacity + translateY de 0 a 1 segundo com easing premium
import { useCurrentFrame, interpolate, Easing } from "remotion";

export const EnterCard = () => {
  const frame = useCurrentFrame();
  const opacity = interpolate(frame, [0, 30], [0, 1], {
    easing: Easing.bezier(0.16, 1, 0.3, 1) // mesmo easing do print
  });
  return <div style={{ opacity, aspectRatio: '3/4' }}>EPI-USE</div>;
};
